# Todo App (React + Framer Motion)

A **beautiful, animated Todo App** with filtering, dark mode, and smooth transitions. Built with **React**, **Framer Motion**, and deployed on **Vercel**.

Live Demo: https://todo-react-nine-gamma.vercel.app/

---

## Features

- Add / Delete / Complete tasks  
- Filter: **All / Active / Completed**  
- Smooth **animations** on add/remove (Framer Motion)  
- **Dark / Light** theme toggle  
- Data saved in **localStorage**  
- Fully responsive  
- Deployed with **Vercel**

---

## Tech Stack

| Technology       | Purpose                     |
|------------------|-----------------------------|
| React            | UI & State Management       |
| Framer Motion    | Animations & Transitions    |
| Vite             | Fast Build Tool             |
| Vercel           | Hosting & Deployment        |
| localStorage     | Persist Todos               |
